Dtella-BPHC

Dtella@BITS Pilani, Hyderabad: A decentralized peer-to-peer file sharing network using the direct connect protocol and restricted to BPHC IPs.

    Dtella is a decentralized direct connect hub used for file sharing. Because it's able to restrict file sharing to a specific IP range, Dtella is ideal for usage at low-latency campus networks where upload limits are imposed for external traffic. While we do not currently have any upload limits, Dtella is perfect for BPHC.

    Windows install instructions

    • Download the .exe.zip from the releases page above.
    • Install Dtella and run it. Add it to your startup applications to make things easy.
    • Open your DC client and add a new favourite/bookmark.
      1. Name: infinity@bphc
      2. Address: 127.0.0.1:7314

    OSX install instructions

    • Download the .dmg from the releases page above.
    • Open the dmg and copy the Dtella.app to your Applications folder.
    • Run Dtella just as you would any other application. It runs in the background.
    • Open your DC client and add a new favourite/bookmark.
      1. Name: infinity@bphc
      2. Address: 127.0.0.1:7314

    Linux install instructions

    • Grab the latest tar/zip snapshot from above.
    • Untar the source and move to your home directory
    
$ tar -xzf dtella-bphc-*.tar.gz 
$ cd dtella-bphc-* 
$ mkdir ~/.dtella 
$ mv <extracted folder>/* ~/.dtella
    • Create a script to launch dtella
    
$ cd ~ 
$ vim dtella
    • Paste this and save.
    
#!/bin/bash 
python2 ~/.dtella/dtella.py &
    • Make the script executable
    $ chmod +x dtella
    • Launch dtella
    ~/.dtella
    • Most linux systems have python preinstalled, but if the above gives errors - you will need to install some packages.
    $ sudo apt-get install python-twisted python-pycrypto
(or)
$ sudo apt-get install python-twisted python-pycryptopp
    • Open your DC client and add a new favourite/bookmark.
      1. Name: infinity@bphc
      2. Address: 127.0.0.1:7314

    Port Forward & Firewall Instructions

    Dtella needs a UDP port for peer communication, and DC++ needs a TCP port for file transfers. If you have a firewall or router that's blocking ports, then check out these guides: http://wiki.dtella.org/wiki/Firewall_Setup_Guide
    http://wiki.dtella.org/wiki/Router_Setup_Guide

    Share

    Once you're on, you'll have to share some files unless you just want to chat. In DC++, go here to get started: File > Settings > Sharing

    Getting help within Dtella

    Type !help into the main chat to see a list of Dtella commands.
